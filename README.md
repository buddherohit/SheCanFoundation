# She Can Foundation Contact Form

A modern, responsive, and user-friendly contact form built for the **She Can Foundation Internship Task** using HTML, CSS, and JavaScript.

## 📌 Overview

This project provides a clean and professional interface that allows users to submit their:

* Full Name
* Email Address
* Message

Upon successful submission, a confirmation message is displayed without reloading the page.

## 🚀 Features

* Modern and responsive UI design
* Mobile-friendly layout
* HTML5 form validation
* Interactive success message
* Clean and organized code structure
* Separate HTML, CSS, and JavaScript files

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## ▶️ Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/she-can-foundation-contact-form.git
```

2. Navigate to the project folder:

```bash
cd she-can-foundation-contact-form
```

3. Open `index.html` in your browser or run it using VS Code Live Server.
